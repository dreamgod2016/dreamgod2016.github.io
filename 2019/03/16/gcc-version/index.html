GCC、G++ versions

发表于 2019-03-16

Change the default version of gcc and g++

  1. 查看所安装的gcc：ls /usr/bin/gcc* -l
  2. 删除已有的软链接：sudo rm /usr/bin/gcc
  3. 建立新的：sudo ln -s /usr/bin/gcc-4.4 /usr/bin/gcc
