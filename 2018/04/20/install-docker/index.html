在Windows上安装Docker ToolBox

发表于 2018-04-20 | 更新于: 2018-04-20

想在Win10 Home的电脑上安装Docker，然而现在的Docker for windows只支持Pro版本的，只能用另一个办法啦。

1. 下载Docker ToolBox

直接戳链接：https://docs.docker.com/toolbox/overview/#ready-to-get-started

然后就下载+默认的安装。

Docker ToolBox是官方为不满足要求的Mac和Windows提供的解决方案，本质上是运行在虚拟机上的Docker。

2. 启动虚拟机

这里需要下载一个boot2docker的镜像文件，但是Terminal下载实在是太太太慢了，我们可以把文件下载下来直接存在指定的文件夹下。

目录一般是：C:\Users\XXXX\.docker\machine\cache,XXXX是电脑用户名。

文件是从github上下载下来的，时间不同版本号也不同，根据Terminal里的提示单独下载下来~

配置的过程中可能会有bug，不过一般会自己修复的~

看到这只可爱的鲸鱼我们就可以用docker做自己的事啦~
